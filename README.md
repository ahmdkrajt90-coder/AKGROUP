# AK Group Real Estate - Istanbul

موقع ويب احترافي لشركة AK Group للعقارات في اسطنبول.

## المميزات
- تصميم عصري وأنيق
- دعم كامل للعربية (RTL)
- موضع تفاعلي
- معلومات العقارات والفريق

## النشر على Vercel

### المتطلبات
- حساب Vercel
- Git و GitHub

### خطوات النشر:

1. **دفع المشروع إلى GitHub**
```bash
git remote add origin https://github.com/YOUR_USERNAME/ak-group-real-estate.git
git branch -M main
git push -u origin main
```

2. **النشر على Vercel**
- اذهب إلى https://vercel.com
- انقر على "New Project"
- اختر repository من GitHub
- انقر على "Deploy"

### التطوير المحلي
```bash
npm run dev
```
ثم افتح http://localhost:3000
